## 4. Custom cursor (NEW: frontend/src/cursor.js)

- Export `initCursor(canvas)` and `setCursorState(state, color?)`.
- Single absolutely-positioned DOM div (cheaper than a THREE sprite) that follows `pointermove` on the canvas. Toggle state with `data-attribute` + CSS transitions.
- Four states:
  - `idle` — pearl droplet (drifts 1px up/down)
  - `hold` — brick tinted by held piece colour
  - `snap` — leaf-green pulsing halo + checkmark SVG
  - `reject` — apple-red shaking disc + ×
- Wire from `interaction.js` (read-only signals; do not change math):
  - On `pointermove` while `getHeldPieceId() !== null`: state = `hold`; switch to `snap` if current `_targetPos` is snapped (reuse existing snapped flag in `_updatePreviewPosition`)
  - In `_rejectPlacement`: `setCursorState('reject')` for 300ms then return
  - On confirm: flash `snap` → fade to `idle` over 200ms
- Hide OS crosshair: `canvas.style.cursor = 'none'` when sprite visible.
