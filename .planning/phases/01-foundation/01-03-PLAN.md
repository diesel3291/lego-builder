Set JSON schema (v1) — every field listed here is required:
```
{
  "schemaVersion": 1,
  "id": "string (kebab-case, matches filename without .json)",
  "name": "string",
  "description": "string",
  "pieceCount": integer,
  "steps": [
    {
      "stepNumber": integer (1-indexed, sequential, no gaps),
      "description": "string",
      "pieces": [
        {
          "id": "string (unique within set)",
          "type": "one of 13 BRICK_TYPES",
          "color": "CSS hex string",
          "gridX": integer,
          "gridZ": integer,
          "layer": integer (0 = baseplate surface),
          "rotation": integer (0, 90, 180, or 270 only)
        }
      ]
    }
  ]
}
```

Valid piece types (must match exactly):
  brick-1x1, brick-1x2, brick-1x3, brick-1x4,
  brick-2x2, brick-2x3, brick-2x4,
  plate-1x1, plate-1x2, plate-1x4, plate-2x2, plate-2x4,
  slope-2x1, slope-2x2

Coordinate system (from grid.js):
  - gridX: stud column (positive = right)
  - gridZ: stud row (positive = away from camera)
  - layer: 0 = resting on baseplate surface; +1 per brick height (bricks) or plate height (plates)
  - World conversion: X = gridX * 8, Z = gridZ * 8, Y_brick = layer * 9.6, Y_plate = layer * 3.2

Flask validator rules (from app.py validate_set()):
  - All listed required fields must be present
  - type must be in VALID_TYPES set
  - rotation must be in {0, 90, 180, 270}
  - gridX, gridZ, layer must pass isinstance(x, int) — JSON floats (1.0) will be rejected

        data = resp.get_json()
        assert isinstance(data, list)
        ids = {s['id'] for s in data}
        assert 'mini-rocket' in ids

        resp = client.get('/api/sets')
        data = resp.get_json()
        for item in data:
            assert 'id' in item
            assert 'name' in item
            assert 'description' in item

        resp = client.get('/api/sets/mini-rocket')
        assert resp.status_code == 200
        data = resp.get_json()
        assert data['id'] == 'mini-rocket'
        assert data['schemaVersion'] == 1
        assert isinstance(data['steps'], list)
        assert len(data['steps']) > 0

    def test_color_steps_has_100_pieces(self, client):
        resp = client.get('/api/sets/color-steps')
        assert resp.status_code == 200
        data = resp.get_json()
        total = sum(len(s['pieces']) for s in data['steps'])
        assert total == 100

    def test_all_pieces_have_integer_coords(self, client):
        resp = client.get('/api/sets')
        for item in resp.get_json():
            data = client.get(f'/api/sets/{item["id"]}').get_json()
            for step in data['steps']:
                for piece in step['pieces']:
                    assert isinstance(piece['gridX'], int), \
                        f"{item['id']}: gridX is not int: {piece['gridX']!r}"
                    assert isinstance(piece['gridZ'], int), \
                        f"{item['id']}: gridZ is not int: {piece['gridZ']!r}"
                    assert isinstance(piece['layer'], int), \
                        f"{item['id']}: layer is not int: {piece['layer']!r}"

    def test_all_rotations_valid(self, client):
        valid = {0, 90, 180, 270}
        resp = client.get('/api/sets')
        for item in resp.get_json():
            data = client.get(f'/api/sets/{item["id"]}').get_json()
            for step in data['steps']:
                for piece in step['pieces']:
                    assert piece['rotation'] in valid, \
                        f"{item['id']}: invalid rotation {piece['rotation']}"

    def test_piece_counts_match_metadata(self, client):
        resp = client.get('/api/sets')
        catalogue = {s['id']: s['pieceCount'] for s in resp.get_json()}
        for set_id, expected_count in catalogue.items():
            data = client.get(f'/api/sets/{set_id}').get_json()
            actual = sum(len(s['pieces']) for s in data['steps'])
            assert actual == expected_count, \
                f"{set_id}: pieceCount={expected_count} but actual pieces={actual}"

        resp = client.get('/api/sets')
