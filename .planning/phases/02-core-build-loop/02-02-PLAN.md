Create `frontend/src/interaction.js` — the central interaction controller.

**Imports:**
```javascript
import * as THREE from 'three';
import gsap from 'gsap';
import { getScene, getCamera, getRenderer } from './scene.js';
import { getGeometry } from './geometry.js';
import { gridToWorld } from './grid.js';
import {
  getHeldPieceId, getCurrentStep, placeBrick,
  isStepComplete, advanceStep, releasePiece, isBuildComplete, getPlacedThisStep
} from './state.js';
import { getGhostMeshes, hideGhost, showStepGhosts, hideAllGhosts } from './ghost.js';
```

**Module-level private state:**
```javascript
let _pointerDownPos = { x: 0, y: 0 };
const CLICK_THRESHOLD_PX = 5;
const _raycaster = new THREE.Raycaster();
const _ndc = new THREE.Vector2();
const _placedMeshes = [];   // Array of THREE.Mesh — all placed opaque bricks
let _onStepAdvance = null;  // callback: called when step advances (for tray/hud re-render)
let _onBuildComplete = null; // callback: called when build finishes
```

**Exported functions:**

1. `initInteraction({ onStepAdvance, onBuildComplete })` — Called once from main.js. Stores the callbacks. Attaches event listeners to the canvas (`getRenderer().domElement`):

   **pointerdown handler:**
   ```javascript
   canvas.addEventListener('pointerdown', (e) => {
     _pointerDownPos = { x: e.clientX, y: e.clientY };
   });
   ```

   **pointerup handler:**
   ```javascript
   canvas.addEventListener('pointerup', (e) => {
     const dx = e.clientX - _pointerDownPos.x;
     const dy = e.clientY - _pointerDownPos.y;
     const dist = Math.sqrt(dx * dx + dy * dy);
     if (dist < CLICK_THRESHOLD_PX) {
       _handleClick(e);
     }
   });
   ```

   **Cursor state management:** Listen for `pointermove` on canvas. If `getHeldPieceId()` is not null, set `canvas.style.cursor = 'crosshair'`. If null, set `canvas.style.cursor = 'default'`.

2. `getPlacedMeshes()` — Returns `_placedMeshes` array. Needed by future phases for undo.

**Private function `_handleClick(event)`:**

```
1. If getHeldPieceId() is null, return early (no piece held — click on canvas does nothing)
2. Compute NDC:
   ndc.x = (event.clientX / window.innerWidth) * 2 - 1;
   ndc.y = -(event.clientY / window.innerHeight) * 2 + 1;
   (Pitfall 4: always use window.innerWidth/Height live, never cached)
3. raycaster.setFromCamera(ndc, getCamera())
4. Build targets array: [...getGhostMeshes()]
   (We raycast against ghosts only — user "clicks the ghost to confirm" per RESEARCH.md Open Question 3)
5. const hits = raycaster.intersectObjects(targets)
6. If no hits, call _rejectPlacement() and return
7. const hitGhost = hits[0].object
8. const pieceId = hitGhost.userData.ghostPieceId
9. Find the piece in getCurrentStep().pieces where piece.id === pieceId
10. Check if pieceId matches getHeldPieceId() — user must have selected this piece in tray
    If mismatch: _rejectPlacement(hitGhost) and return
11. Call _confirmPlacement(piece, hitGhost)
```

**Private function `_confirmPlacement(piece, ghostMesh)`:**

```
1. Create opaque brick mesh:
   const geometry = getGeometry(piece.type);
   const material = new THREE.MeshStandardMaterial({ color: piece.color, roughness: 0.6, metalness: 0.1 });
   const mesh = new THREE.Mesh(geometry, material);
   mesh.position.copy(gridToWorld(piece.gridX, piece.gridZ, piece.layer, piece.type));
   mesh.rotation.y = THREE.MathUtils.degToRad(piece.rotation);
   mesh.castShadow = true;
   mesh.receiveShadow = true;
   mesh.userData.pieceId = piece.id;
   getScene().add(mesh);
   _placedMeshes.push(mesh);

2. Success flash — tween material color from green to piece color:
   const targetColor = new THREE.Color(piece.color);
   mesh.material.color.set(0x4caf50);  // success green from UI-SPEC
   gsap.to(mesh.material.color, {
     r: targetColor.r, g: targetColor.g, b: targetColor.b,
     duration: 0.4, ease: 'power2.out'
   });

3. Remove the ghost for this piece:
   hideGhost(piece.id);

4. Update state:
   placeBrick(piece);    // marks cells occupied
   releasePiece();       // clear held piece

5. Check step completion:
   if (isStepComplete()) {
     if (isBuildComplete()) {
       hideAllGhosts();
       if (_onBuildComplete) _onBuildComplete();
     } else {
       advanceStep();
       const nextStep = getCurrentStep();
       if (nextStep) showStepGhosts(nextStep);
       if (_onStepAdvance) _onStepAdvance();
     }
   }
   // If step is NOT complete (multi-piece step), ghosts for remaining pieces stay visible
```

**Private function `_rejectPlacement(ghostMesh)`:**

```
If ghostMesh is provided (user clicked a ghost but for wrong piece):
  const origColor = ghostMesh.material.color.clone();
  ghostMesh.material.color.set(0xe53935);  // rejection red from UI-SPEC
  gsap.to(ghostMesh.material.color, {
    r: origColor.r, g: origColor.g, b: origColor.b,
    duration: 0.3, ease: 'power1.inOut'
  });

If no ghostMesh (user clicked empty space): no visual feedback needed
```

**GSAP fallback note:** If GSAP cannot tween THREE.Color properties directly (Assumption A1 from RESEARCH.md), replace with this pattern:
```javascript
gsap.to({ t: 0 }, {
  t: 1, duration: 0.4, ease: 'power2.out',
  onUpdate: function() {
    mesh.material.color.lerpColors(startColor, endColor, this.targets()[0].t);
  }
});
```
Try the direct `gsap.to(material.color, {...})` approach first. If it does not produce visible color change at runtime, switch to the fallback.

**Critical design rules:**
- NDC must be computed fresh on every click from `window.innerWidth/Height` — never cache (Pitfall 4)
- Raycast only against ghost meshes (not baseplate) — user confirms by clicking the ghost (RESEARCH.md OQ3)
- Never dispose geometry from getGeometry() — only create new materials per placed brick
- Use `castShadow: true` and `receiveShadow: true` on placed bricks (matches scene.js shadow setup)
- Callbacks (`onStepAdvance`, `onBuildComplete`) allow main.js to trigger tray/hud updates without circular imports
